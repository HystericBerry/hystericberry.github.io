Arxiv Search Engine:
Roles - Lead Project Manager & Software Engineer

This search engine is capable of parsing document files formatted in JSON and evaluating relevant documents. Cornell University's online scientific library known as Arxiv. This ArxivSearchEngine is currently of:

  • Fastest Indexing Time (of entire class) - 36000+ articles in 1.6 minutes
  • Processing AND, OR, AND NOT, PHRASE and Wild Card (*) search Queries
  • Variable Byte Encoding Compression
  • Gap Encoding Compression
  • Querying from Auxilary and Disk Indicies
  • Spelling Correction using k-grams and levenshtein edit distance
  • Document Relevancy using Rocchio Classification and Vector Space Models
  • Document Relevancy using Naive Bayesian Document Relevancy
  • Test Driven Development using Google Test C++
